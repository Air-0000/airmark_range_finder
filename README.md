# 航标测距仪 (Airmark Range Finder)

基于 STM32F407 的嵌入式航标测距仪项目。

## 技术栈

- **MCU**: STM32F407 (ARM Cortex-M4)
- **语言**: C
- **开发环境**: STM32CubeMX, Keil MDK

## 目录结构

```
airmark_range_finder/
├── Core/          # 核心代码（主程序、中断、外设驱动）
├── Drivers/       # 官方驱动库（HAL/LL）
├── Middlewares/   # 中间件
└── User_Code/     # 用户自定义代码
```
